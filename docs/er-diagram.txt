  ┌─────────────────────────────────────────────────────────────────────────────┐
  │                         FLUJO DE INSCRIPCIÓN                               │
  └─────────────────────────────────────────────────────────────────────────────┘

  Alumno elige modalidad
          │
          ▼
  Se crea DetalleProgramaAlumno (estado="postulante")
          │
          ▼
  Se genera ControlDocumentacion por CADA requisito de esa modalidad
          │
          │  (query: JOIN Requisito ↔ ModalidadRequisito
          │   filtrando por id_modalidad_academica)
          │
          ▼
  Revisión: pendiente → entregado → aceptado/rechazado
          │
          ▼
  Si TODOS los obligatorios = "aceptado" → estado = "inscrito"
